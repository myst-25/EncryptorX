EncryptorX

Simple Encryption

Like Morse code for text — encrypt with a password, share the code, decrypt with the same password

Encrypt

How it works: Enter your password and text, click Encrypt, then share the encrypted code with others.

Minimum 8 characters

Decrypt

How it works: Paste the encrypted code, enter the password, and click Decrypt to read the message.

Same password used during encryption

🔒 All encryption is done in your browser. Nothing is sent to any server.
